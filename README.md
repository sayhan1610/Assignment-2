# Assignment 2: Conversion Tool

A Flask web application for converting numbers, IP addresses, and RGB colors between binary, decimal, and hexadecimal representations.

## Features

- Number conversions: Binary ↔ Decimal ↔ Hexadecimal
- IP Address conversions: Binary ↔ Decimal ↔ Hexadecimal
- RGB Color conversions: Binary ↔ Decimal ↔ Hexadecimal

## Installation

1. Ensure Python is installed.
2. Install Flask: `pip install flask`
3. Run the application: `python app.py`
4. Open http://127.0.0.1:5000 in your browser.

## Structure

- `app.py`: Main Flask application
- `functions/`: Directory containing conversion functions
- `templates/index.html`: Web interface
